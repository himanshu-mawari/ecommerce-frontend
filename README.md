## E-commerce Application
- A full-stack e-commerce application that digitizes product discovery, cart management, order processing, and secure online payments through a scalable web architecture
## Why It Exists

- Traditional retail is limited by physical location, fixed store hours, restricted inventory visibility, manual billing, and limited payment options.

- This E-commerce application removes those constraints by digitizing product discovery, real-time inventory tracking, automated checkout, and multi-mode payment processing through an integrated software system.
